About ASCII Art Web

Description

ASCII Art Web is a web-based application that allows users to convert text into ASCII art using different banners such as "shadow", "standard", and "thinkertoy". Users can input text and select their preferred banner to generate and display the ASCII art.

Authors

  • Author 1: [Valeria Muhembele]
  • Author 2: [Sheilla Juma]

Usage

  • Clone the repository: git clone [https://learn.zone01kisumu.ke/git/vmuhembe/ascii-art-web-export-file.git]
  • Navigate to the project directory: cd ascii-art-web
  • Run the server: go run main.go
  • Open your browser and go to: http://localhost:3000

Implementation Details

The application is built using Go's HTTP package to create a web server that handles GET and POST requests. HTML templates are used for rendering the web pages, and the ASCII art conversion is performed on the server side based on user input. CSS is used for styling the web pages, ensuring they are responsive and user-friendly.

The main algorithm involves parsing the input text and converting each character into its corresponding ASCII art representation based on the selected banner.
